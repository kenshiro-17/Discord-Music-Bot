Slash Commands

Easy-to-use commands with Discord's native interface

Playback

/play <query> Play a song from YouTube, Spotify, or search
/pause Pause the current song
/resume Resume playback
/skip Skip to the next song
/stop Stop playback and clear queue

Queue

/queue View the current song queue
/nowplaying Show currently playing song
/remove <position> Remove a song from queue
/jump <position> Jump to a specific song
/shuffle Shuffle the queue
/clear Clear the entire queue

Settings

/volume <0-200> Set playback volume
/loop <mode> Set loop mode (off/song/queue)
